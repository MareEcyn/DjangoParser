# Parser

Система состоит из клиента (Django приложение), сервера, и html-парсера.

### How it works

Пользователь добавляет запросы в базу (url, time). Статичный планировщик задач посылает эти запросы на сервер. Парсер, живуший на сервере, обрабатывает полученный запрос и отсылает результат обратно. Полученный ответ в свою очередь обрабатывает планировщик, и, при необходимости, записывает результаты в базу Django.

## Install

Req: Python 3.7+

```
pip install -r requirements.txt
```

## Use

In project root:

```
python run.py
```
